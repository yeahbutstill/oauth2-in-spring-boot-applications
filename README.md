# oauth2-in-spring-boot-applications

## Download and Install [keycloak](https://www.keycloak.org/getting-started/getting-started-zip)
- run keycloack
- Create admin user
- Login to the admin console
- Create realm
- Create user
- Secure client app

## Run app
```shell
mvn clean install spring-boot:run
```
